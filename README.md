# HR Analytics Dashboard

## Project Overview
Developed an interactive HR Analytics Dashboard in Power BI to analyze employee attrition, workforce demographics, and key HR metrics. Utilized Power Query for data transformation, DAX for KPI calculations, and interactive visualizations to identify workforce trends, support retention strategies, and enable data-driven HR decision-making.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

## Key Features
- Employee Attrition Analysis
- Workforce Demographics Analysis
- Department-wise Insights
- Interactive Filters and Slicers
- KPI Tracking and Reporting
- Detailed Employee Analysis
